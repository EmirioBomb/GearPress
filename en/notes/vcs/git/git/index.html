📌 What is Git

Note

Git is a distributed version control system(VCS) used to track changes in files and support collaborative software development.

It helps developers save versions, view history, and work together without conflicts.

🔁 Workflow

🖥️ Cheat Sheet

CommandDescription
🟢 Initialization 🟢
git initInitialize a new Git repository
git clone <repository-url>Clone a remote repository to local
🟡 Configuration 🟡
git config --listView all Git configuration
git config --global user.name <your name>set global username
git config --global user.email <your@email.com>set global email
git config --global commit.gpgsign trueEnable GPG commit signing in Git
git config --global https.proxy 127.0.0.1:7890HTTPs proxy for Git traffic
git config --global http.proxy 127.0.0.1:7890HTTP proxy for Git traffic
🟣 Stage 🟣
git add <file>Stage a file
git add .Stage all changes
git reset <file>Unstage a file Git （< 2.23）
git restore --staged <file>Remove file from staging area Git （>= 2.23）
🔴 Commit 🔴
git commit -m "message"Create a commit
git commit -am "message"Same to git add . && git commit
git commit --amend -m "new message"Edit last commit message
🟠 Branch 🟠
git branchList branches
git branch <branch-name>Create branch
git switch <branch-name>Switch branch
git switch -c <branch-name>Create and switch branch
git merge <branch-name>Merge branch into current branch
git branch -d <branch-name>Delete branch
🔵 Remote Repository 🔵
git remote -vView remote repositories
git remote add origin <url>Add remote repository
git push origin mainPush commits to remote
git pull origin mainPull and merge remote changes
git fetchFetch remote updates only
🟩 Recovery 🟩
git restore <file>Discard local changes
git reset --soft HEAD~1Undo last commit but keep changes
git reset --hard HEAD~1Remove commit and changes permanently
git revert <commit>Create reverse commit
🟥 Stash 🟥
git stashTemporarily save changes
git stash listView stash list
git stash popRestore latest stash
git stash clearDelete all stash records
🟧 Tag 🟧
git tagList tags
git tag <tag-name>Create tag
git push origin <tag-name>Push tag to remote
